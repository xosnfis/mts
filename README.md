# MWS GPTHub

Умный AI-чат на базе [Open WebUI](https://github.com/open-webui/open-webui) с роутером, который автоматически выбирает нужную модель под задачу: текст, код, изображения, голос, презентации, веб-поиск и память между чатами.

---

## Требования

- [Docker Desktop](https://www.docker.com/products/docker-desktop/) (Windows / macOS / Linux)
- Git (опционально, если клонируешь репозиторий)

---

## Быстрый старт

### 1. Клонируй или распакуй проект

```bash
git clone <url-репозитория>
cd <папка-проекта>
```

Или просто распакуй архив и открой терминал в папке проекта.

### 2. Запусти

```bash
docker compose up --build -d
```

Первый запуск займёт несколько минут — Docker скачает образы и соберёт контейнер.

### 3. Открой в браузере

- Чат: [http://localhost:3000](http://localhost:3000)
- Просмотр БД (SQLite Web): [http://localhost:8082](http://localhost:8082)

При первом входе создай аккаунт администратора.

---

## Что умеет роутер

Роутер автоматически определяет тип запроса и выбирает модель:

| Тип запроса | Модель |
|---|---|
| Обычный текст / вопросы | `llama-3.3-70b-instruct` |
| Код / программирование | `qwen3-coder-480b-a35b` |
| Изображения (анализ) | `qwen2.5-vl-72b` |
| Голосовые сообщения (STT) | `whisper-turbo-local` |
| Память / персонализация | `mts-anya` |
| Эмбеддинги / RAG | `bge-m3` |

Дополнительные возможности:
- Генерация изображений по тексту
- Создание презентаций (.pptx) по запросу
- Веб-поиск и глубокий ресёрч
- Долгосрочная память между чатами (можно отключить в настройках)

---

## Управление

Остановить:
```bash
docker compose down
```

Перезапустить после изменения `.py` файлов пайплайнов:
```bash
docker compose up --build -d
```

Посмотреть логи:
```bash
docker compose logs -f
```

## Частые проблемы

**Порт 3000 занят** — измени в `docker-compose.yml` строку `"3000:8080"` на другой порт, например `"3001:8080"`.

**Docker не запускается** — убедись, что Docker Desktop запущен перед выполнением команд.

**Контейнер падает** — проверь логи: `docker compose logs -f open-webui`
